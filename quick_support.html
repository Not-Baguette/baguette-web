- How to disable Tabsv2?

Step 1: Get Vendetta/Revenge/Revenge

In order to get into the settings, you need to get Vendetta/Revenge. Vendetta/Revenge is a mod for discord that allows you to change settings that are not normally accessible. It is also a mod that allows you to change the UI of discord. I must warn you that this is against discord's TOS, so use it at your own risk. You can get Revenge here or Vendetta .

Step 2: Get Experiments Plugin

You can find experiments plugin in the Vendetta discord server. You can find the discord server here. Once you have joined the server, go to the #plugins channel and download the plugin by long pressing it and clicking Install plugin. Once you have downloaded the plugin, proceed to the next step.

Step 3: Enable TabsV2 toggles

Go to settings, Experiments, and then search for these two options below. If you couldn't find these, you'll have to downgrade your discord

Once you have found it, change both of them to treatment 1:show and treatment 1:control. You should now be able to find the option on step 3.

Step 4: Disable it in settings

Disable Tabs UI Experiment in the settings page as shown above. (You will need to scroll wayyy down) However, if you couldn't find it, Please refer to step 3.

- Extras 4: How to Downgrade Vendetta/Revenge

You can check your Vendetta/Revenge manager by opening the app and checking if there's an 'i' on the top right corner (1.1.0 or below) or a gear icon there (1.1.1 or above)

I am using Vendetta/Revenge manager 1.1.1 or above
I am using Vendetta/Revenge manager 1.1.0 or below

First, go to Vendetta/Revenge manager, and click the gear icon on top right.

Then, click the 'about' option with the i icon on the bottom.

Then, click the version text around ten times to access the developer options.

There should be a pop-up confirming so, after that, click settings on bottom right of the app. Now go to 'developer options' above 'about' option on the settings. And then press on the "Discord Version" entrybox and write the version you wanted

after that go back to the main page and click uninstall so we can re-install it again and you're done!
