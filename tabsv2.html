How to disable Tabsv2 on discord

Tested on latest versions like 206.16, 207.23, 208.17, 209.14, and 211.11

- What is Tabsv2?

Tabsv2 is a new UI for discord mobile that is currently being rolled out. It is a new way to view your servers and DMs. It has been tested for many months before it's rollout starting around 6 december 2023. Despite seemingly almost common knowledge, the rollout seemed to cause a minor backlash from the community.

- How to disable Tabsv2?

Step 1: Get Vendetta

In order to get into the settings, you need to get Vendetta. Vendetta is a mod for discord that allows you to change settings that are not normally accessible. It is also a mod that allows you to change the UI of discord. I must warn you that this is against discord's TOS, so use it at your own risk. You can get Vendetta here.

Step 2: Get Experiments Plugin

You can find experiments plugin in the vendetta discord server. You can find the discord server here. Once you have joined the server, go to the #plugins channel and download the plugin by long pressing it and clicking Install plugin. Once you have downloaded the plugin, proceed to the next step.

Step 3: Disable it in settings

Disable Tabs UI Experiment in the settings page as shown above. (You will need to scroll wayyy down) However, if you couldn't find it, Please refer to step 4.

Step 4: Couldn't find it?

Go to settings, Experiments, and then search for these two options below

Once you have found it, change both of them to treatment 1:show and treatment 1:control. You should now be able to find the option on step 3.

- Extras 2: How to Downgrade Vendetta

UPDATE: 207.xx and above is fixed. If you're here to downgrade due to the issue with 207 and higher, you are no longer needed to, feel free to update to the latest version of vendetta.

Information: You can see the version comparison list at below or here!

First, go to vendetta manager, and click the 'i' button on top.

Then, click the version text around ten times to access the developer options.

There should be a pop-up confirming so, after that, click settings on bottom right of the app

Once you've clicked it, press on the "Discord Version" entrybox and write the version you wanted. after that go back to the main page and click uninstall so we can re-install it again and you're done!
